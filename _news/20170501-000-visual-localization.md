Recent work on using semantic cues to localize a vehicle will be presented at <a href="http://www.icra2017.org/">the 2017 IEEE International Conference on Robotics and Automation (ICRA)</a>.
This is joint work with <a href="http://people.csail.mit.edu/weichium/">Wei-Chui Ma</a>, <a href="http://www.cs.toronto.edu/~slwang/">Shenlong Wang</a>, <a href="http://www.cs.utoronto.ca/~fidler/">Sanja Fidler</a> and <a href="http://www.cs.utoronto.ca/~urtasun/">Raquel Urtasun</a>.
You can find the paper <a href="http://arxiv.org/abs/1606.07415">here</a> and a video of some of the results <a href="https://youtu.be/tOVWptik0qI">here</a>.
